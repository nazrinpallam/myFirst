Types of Selectors:

1.Element Selector

It selects HTML elements by name.

2. ID Selector

It uses ID attribute of HTML to select specific element. ID is always unique within a page. Its weitten with (#) character

3.Class Selector

It selects HTML elemnt with specific class attribute. Its used with period (.) character.

4.Universal Selector

It selects all elemnts on the pages.Its used with (*) character.

5.Group Selector

Its used to select all the elements with the same style definitions. Its used to minimise the code. Commas are used to separate each selector.
